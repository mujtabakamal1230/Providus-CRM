# Claude Skills

This directory contains project-local Claude skills.

- `skills/providus-crm-development` is tailored to this Next.js 15 project and its design system.
- The remaining skills were copied from [`addyosmani/agent-skills`](https://github.com/addyosmani/agent-skills) to provide reusable engineering workflows for planning, implementation, testing, debugging, review, security, performance, and shipping.

Keep upstream skill updates small and intentional. When changing project behavior, prefer updating `skills/providus-crm-development/SKILL.md` so Claude keeps this repository's design system and conventions front and center.
